# [Hacettepe Book Manager](https://hacettepebookmanager.herokuapp.com)
*****

このwebアプリはあなたの読書を支援します!!!  
具体的にはハジェテぺ大学が考案した知識定着率を向上させる17の質問に答えてもらいます  
始めに全て埋めるのではなく気付きが合った都度答えていきましょう!  
そして読書においてのインプット・アウトプットをより良質なものに一緒にしていきましょう
